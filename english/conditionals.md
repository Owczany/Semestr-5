# Conditionals

### 0 (zero conditional)

Pattern:
If + present simple, present simple

Example:
1. If it rains, the grass gets wet.
2. If the temperature is low, water freezes.
3. When you don't drink enough water, you are thirsty.
4. When you put sugar in your tea, it becomes sweeter.
5. If you see a red light, stop.

When to use it:
When it's always true. When the result is so obvious.

### 1 (first conditional)
  
Pattern:
IF + PRESENT SIMPLE + ',' + FUTURE SIMPLE (WILL) + VERB

Example:
1. If it is sunny, we will go to the beach.
2. If it rains, we will stay at home.
3. They will come to the party if there are popular kids there.
4. If there is any candy, he will eat all of it.

### 2 (second conditional)

Pattern:
IF + PAST SIMPLE + ',' + WOULD + VERB

Example:
1. If I won a million dollars, I would buy a new car.
2. If I were you, I would quit smoking.
3. They would have another baby if they weren't so focused on their careers.
4. If I were qualified enough, I would look for another job.

1. If I were you, I would sell this car.
